# Código utilizado en el TFM de Iratxe Gil Vivanco

En el archivo Codigo_TFM_Iratxe_Gil_Vivanco se encuentra el código fuente y los resultanos numéricos obtenidos en el TFM de Iratxe Gil Vivanco, titulado "Métodos de Optimización Matemática para la Toma de Decisiones con Múltiples Fuentes de Datos". Toda la base teórica y el planteamiento de los ejemplos se encuentra en la memoria de este.

El código se ha realizado utilizando el lenguaje Python. Se han utilizado las librerias numpy, scipy.stats, matplotlib.pyplot y amplpy. La librería amplpy permite utilizar el programa AMPL, que es un lenguaje de modelado algebraico para optimización matemática que permite describir y resolver problemas de gran escala.
